Get Started Quickly

1. Download Tetrate Istio Distro CLI

curl -sL https://istio.tetratelabs.io/getmesh/install.sh | bashcopy

2. Install Istio Demo

getmesh istioctl install --set profile=democopy

3. Check Tetrate Istio Distro version

getmesh versioncopy

4. Validate config

getmesh config-validatecopy
