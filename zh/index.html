什么是 GetIstio

动词
/getˌis,’tē,ō/

  1. 为 Kubernetes 或应用平台安装和管理经过审核的 Istio。

    在 Kubernetes 集群中安装和更新 Istio 的最佳方法是什么？很简单，你只要 GetIstio！

    类似的

    • service mesh
    • control plane
    • application networking
    • func-e

名词/收益

  1. 简单、开源的 CLI，用于确保特定 Istio 版本的安装和升级。

    我刚刚用 GetIstio 在五分钟内升级到最新的 Istio 版本。

    类似的

    • kubectl
    • istioctl

  2. 上游和 FIPS 兼容的 Istio 分布测试，并针对 EKS、GKE、AKS 等进行加固。

  3. 来自 AWS、Weaveworks、Venafi 等生态系统合作伙伴的验证 Istio 集成和教程的枢纽。

    使用 GetIstio CLI，我在几分钟内就使 AWS 的 ACM 成为我对 Istio mesh 的信任证书提供商。

组合名词

  1. Istio 用户、开发者、教程和技术合作伙伴的社区。

快速开始

1. 下载 GetIstio

curl -sL https://istio.tetratelabs.io/getmesh/install.sh | bash复制

2. 安装 Istio Demo

getmesh istioctl install --set profile=demo复制

3. 检查 GetIstio 版本

getmesh version复制

4. 验证配置

getmesh config-validate复制
